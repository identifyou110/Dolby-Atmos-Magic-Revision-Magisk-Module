# Dolby Atmos Magic Revision Magisk Module

## Descriptions
- An EQ ported by Little Monk reworked
- Powered by Dolby Digital Plus
- Global type soundfx

## Requirements
- Android 4.2 and up
- Magisk installed

## Installation Guide
- Install via Magisk Manager or Recovery
- Reboot

## Tested on
- Android 11 DotOS ROM
- Android 12 AncientOS ROM
